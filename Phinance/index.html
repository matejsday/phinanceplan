Hilfe & Funktionen
◈ Budget
Betrag ändern
Klicke auf die Zahl → Popup öffnet sich → Betrag eingeben → wähle: Nur [Monat], Ab [Monat] (monatlich wiederkehrend), Ganzes Jahr, oder Abbrechen.
Einnahmen hinzufügen
Klicke auf "+ Hinzufügen". Betrag eingeben → Dialog: Nur diesen Monat, Ab diesem Monat (fix), oder Ganzes Jahr.
Ausgabe löschen
Klicke auf × → Dialog: Nur [Monat] (auf 0 setzen), Ab [Monat] (verschwindet ab dann), Aus ganzem Jahr (komplett), oder Abbrechen.
Tags
#fix = Fixkosten (50%-Ziel) · #lifestyle = variable Ausgaben (30%-Ziel) · #sparen = Sparrate (20%-Ziel) · kein Tag = variabel, erscheint nur wenn Betrag eingetragen.
Kategorien verschieben
Halte das ⠿ Symbol links beim Kategorienamen und ziehe die Kategorie an die gewünschte Position.
Schnelleingabe
Klick auf den Gesamtbetrag einer Kategorie → alle Beträge auf einmal eingeben → "Alle Monate speichern".
Notizen
Klicke auf das ✎ Symbol rechts bei einer Ausgabe → Notizfeld öffnet sich direkt in der Zeile.
◇ Optimierung
50/30/20 Regel
Standard: 50% Fixkosten, 30% Lifestyle, 20% Sparen. Klicke auf die farbigen Prozent-Badges um deine eigenen Zielwerte festzulegen. ✓ grün = im Ziel.
Fortschrittsbalken
Zeigt wie viel % deines Einkommens bereits verplant ist. Grün = gut, Gelb = über 80%, Rot = über 100%.
◎ Sparziele
Sparziel erstellen
Klicke auf "+ Neues Sparziel". Name eingeben → verknüpft automatisch mit gleichnamigem #sparen-Eintrag in Ausgaben.
Monatliche Gutschrift
Beim Login wird die Sparrate automatisch zum "Gespart"-Stand addiert für alle vergangenen Monate seit dem letzten Login.
Simulation
Navigiere in die Zukunft → die Karten zeigen den projizierten Stand basierend auf der aktuellen Rate. "accomplished by" zeigt das voraussichtliche Erreichungsdatum.
Karten verschieben
Halte das ⠿ Symbol und ziehe die Karte an die gewünschte Position.
◎ Jahresübersicht
Monatsvergleich
Balkendiagramm zeigt Ausgaben pro Monat. Die goldene Linie = Einkommensniveau. Roter Balken = Ausgaben über Einnahmen.
Jahresbilanz
Berücksichtigt createdAt (ab wann eine Ausgabe gilt) und deletedFrom (ab wann sie weggefallen ist) — nur tatsächlich aktive Monate werden summiert.
◎ Profil & Daten
CSV exportieren
Menü → Profil → Daten exportieren.
Automatische Backups
Jede Änderung wird automatisch gesichert (max. alle 5 Min., bis 30 Stände). Menü → Profil → Backups anzeigen → Wiederherstellen.
Heute-Button
Klicke "heute" neben der Monatsnavigation um sofort zum aktuellen Monat zurückzuspringen.
Mehrere Geräte
Daten werden über Firebase in Echtzeit synchronisiert.
